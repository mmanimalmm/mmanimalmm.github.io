프로젝트 개요

AI 요소와 게임 시스템을 결합한 기획 프로젝트. 기획 단계부터 QA 관점으로 참여해 사용자 경험 흐름과 게임 구조의 잠재적 결함을 사전에 검토. 테스트 가능한 방향으로 기획 완성도를 높이는 데 기여했습니다.

QA 참여 방식

  • 기획서 단계에서 UX 흐름 검토 및 잠재적 결함 사전 식별
  • 게임 루프 구조의 논리적 결함 및 엣지케이스 분석
  • 테스트 가능성(Testability) 관점에서 기획 피드백 제공
  • 팀 내 QA 관점 공유로 기획 완성도 향상 기여

사용 도구

  • Notion — 기획서 작성 및 팀 협업
  • Unity — 프로토타입 검토 참여
  • GitHub — 버전 관리 및 이슈 트래킹
